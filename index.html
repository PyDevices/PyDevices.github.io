PyDevices

One display & graphics stack. Three Python runtimes.

PyDevices is a cross-platform Python stack for displays, input, and graphics. Start from pydisplay's pure-Python drivers, add native modules where you need speed, and reach for LVGL when you want a full widget toolkit — without rewriting the app for each interpreter.

The same ideas run on MicroPython and CircuitPython firmware, on CPython desktops, as an installable PyScript PWA in the browser, and as Android APKs. Explore the product pages below, try the live demos, or jump into the docs when you are ready to wire a board or ship a package.

The stack

Start with pydisplay; add palettes or pdwidgets for color and widgets, or LVGL for a full toolkit.

Pure-Python display, input, and event drivers for MicroPython, CircuitPython, and CPython. The foundation of the stack.

Lightweight widget toolkit for pydisplay — buttons, forms, navigation, and themes without a full LVGL stack.
